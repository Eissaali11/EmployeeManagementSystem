 (محدثة)
1. قاعدة البيانات
إنشاء جدول جديد: mobile_devices

id - المعرف الأساسي
phone_number - رقم الهاتف (مطلوب)
imei - رقم الـ IMEI (مطلوب، فريد)
email - الإيميل المرتبط بالجهاز
employee_id - معرف الموظف (اختياري للربط)
device_model - نوع الجهاز
device_brand - ماركة الجهاز
status - حالة الجهاز (متاح، مرتبط، معطل)
assigned_date - تاريخ الربط
created_at - تاريخ الإنشاء
updated_at - تاريخ التحديث
2. الواجهات المطلوبة
أ. صفحة إدارة الأجهزة المحمولة:

عرض جميع الأجهزة في جدول
فلاتر: الحالة، نوع الجهاز، الموظف المربوط
بحث بالإيميل أو رقم الهاتف أو IMEI
إضافة جهاز جديد يدوياً
تعديل/حذف الأجهزة
ب. صفحة استيراد من Excel:

تحميل ملف Excel يحتوي على: رقم الهاتف، رقم الـ IMEI، الإيميل، نوع الجهاز
معاينة البيانات قبل الاستيراد
معالجة الأخطاء والتكرار في أرقام الـ IMEI
ج. صفحة ربط الأجهزة بالموظفين:

قائمة الأجهزة غير المربوطة
بحث الموظفين وربطهم بالأجهزة
فك الربط عند الحاجة
3. لوحة المعلومات (Dashboard)
إحصائيات سريعة:

إجمالي الأجهزة المسجلة
الأجهزة المربوطة بموظفين
الأجهزة غير المربوطة (متاحة)
الأجهزة المتاحة لموظفين غير نشطين
الجداول التفاعلية:

الأجهزة غير المربوطة مع تفاصيلها (رقم الهاتف، IMEI، الإيميل)
الأجهزة المربوطة بموظفين غير نشطين
آخر العمليات (ربط، فك ربط، إضافة)
4. المسارات والوظائف
المسارات الرئيسية:

/mobile-devices - صفحة إدارة الأجهزة
/mobile-devices/create - إضافة جهاز جديد
/mobile-devices/import - استيراد من Excel
/mobile-devices/assign - ربط الأجهزة بالموظفين
/mobile-devices/dashboard - لوحة المعلومات
الوظائف المطلوبة:

عرض وإدارة الأجهزة (CRUD)
استيراد البيانات من Excel
ربط/فك ربط الأجهزة بالموظفين بناءً على رقم العمل
إحصائيات وتقارير
تصدير البيانات إلى Excel
5. الميزات الإضافية
النظام الذكي:

تنبيهات للأجهزة غير المربوطة
تتبع تاريخ الربط والتغييرات
تقارير شهرية للاستخدام
عرض الأجهزة المتاحة عند تغيير حالة الموظف لغير نشط
الأمان والتدقيق:

تسجيل جميع العمليات في audit log
صلاحيات مختلفة للمستخدمين
حماية بيانات الإيميل وأرقام الـ IMEI
6. التكامل مع النظام الحالي
ربط بجدول الموظفين:

استخدام علاقة many-to-one
ربط الجهاز بالموظف عن طريق رقم العمل
عرض الأجهزة في صفحة تفاصيل الموظف
فلترة حسب حالة الموظف (نشط/غير نشط)
إضافة للقائمة الجانبية:

قسم "الأجهزة المحمولة" مع الصفحات الفرعية
رابط سريع في لوحة المعلومات الرئيسية
7. خصائص الربط
ربط الجهاز بالموظف:

يتم الربط باستخدام رقم العمل للموظف
كل جهاز يمكن ربطه بموظف واحد فقط
عند تغيير حالة الموظف لغير نشط، يظهر جهازه كمتاح للإعادة التخصيص
هذه خطة شاملة ومحدثة تركز على رقم الـ IMEI وليس الإيميل كما طلبت.
